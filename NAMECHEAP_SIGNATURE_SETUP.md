# How to Set Up Email Signature in Namecheap Private Email

## Step-by-Step Instructions

### Step 1: Access Namecheap Private Email Webmail

1. **Open your web browser**
2. **Go to:** https://privateemail.com/
3. **Log in** with:
   - **Email:** hello@zegamedia.com
   - **Password:** Your email password

---

### Step 2: Navigate to Settings

**After logging in, you'll see the webmail interface.**

**Look for the Settings icon** - it's usually:
- A gear icon (⚙️) in the top right corner
- Or in the left sidebar menu
- Or click your name/email in the top right → Settings

**Click on "Settings"**

---

### Step 3: Find Email Signature Section

**In the Settings menu, look for:**
- **"Email"** or **"Mail"** section
- **"Compose"** or **"Composing"** options
- **"Signature"** option

**Click on "Signature"** or navigate to the signature settings

---

### Step 4: Create/Edit Signature

**You'll see a signature editor box.**

**Option B: Rich Text Mode (If HTML doesn't work)**
1. **Use the formatting toolbar** to format text
2. **Type or paste this:**

```
Gage Grace
Founder, ZEGA MEDIA

Email: hello@zegamedia.com
Web: www.zegamedia.com

ZEGA MEDIA | Web Design Agency
```

3. **Format it:**
   - Make "Gage Grace" bold and larger
   - Make links clickable (select text → Insert Link)
   - Add colors if possible

---

### Step 5: Enable Signature

**Look for checkboxes or options like:**
- ☑ **"Add signature to new messages"**
- ☑ **"Add signature to replies"**
- ☑ **"Add signature to forwards"**

**Check the boxes** for when you want the signature to appear.

---

### Step 6: Save

1. **Scroll down** to find the **"Save"** or **"Apply"** button
2. **Click "Save"**
3. **You should see a success message** like "Settings saved" or "Signature updated"

---

### Step 7: Test Your Signature

1. **Click "Compose"** or **"New Message"** to create a new email
2. **Check that your signature appears** at the bottom
3. **Test the links:**
   - Click the email link
   - Click the website link
4. **Send yourself a test email** to see how it looks

---

## Alternative: If You Can't Find Signature Settings

**Some Namecheap Private Email interfaces are different. Try these:**

### Method 1: Through Roundcube (Common Interface)
1. **Look for "Settings"** in the left sidebar
2. **Click "Identities"** or **"Compose"**
3. **Find "Signature"** section
4. **Edit and save**

### Method 2: Through Horde (Alternative Interface)
1. **Look for "Preferences"** in the top menu
2. **Click "Mail"** → **"Compose"**
3. **Find "Signature"** option
4. **Edit and save**

## Troubleshooting

### Signature Not Appearing?
- Make sure you **saved** the settings
- Check that **"Add signature to new messages"** is enabled
- Try logging out and back in
- Clear browser cache and try again

### HTML Not Working?
- Some email clients strip HTML
- Try the plain text version instead
- Use the formatting toolbar to style it manually

### Links Not Clickable?
- Make sure you're using HTML mode
- In rich text mode, manually insert links using the link button
- Test links after adding

### Can't Find Settings?
- Look for "Preferences", "Options", or "Configuration"
- Check both left sidebar and top menu
- Try different webmail interfaces (Roundcube vs Horde)
